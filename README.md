# 🌡️ 기온 비교 웹앱 (같은 날짜 대비 분석)

기상청 일별 기온 데이터를 활용하여  
**특정 날짜가 과거 같은 월–일 대비 얼마나 추웠거나 더웠는지**를  
인터랙티브 그래프(Plotly)로 확인할 수 있는 Streamlit 웹앱입니다.

> 📌 기본 데이터가 탑재되어 있으며,  
> 📌 같은 형식의 CSV를 추가 업로드하면 자동으로 분석에 반영됩니다.

---

## 🔍 주요 기능

- ✅ **기본 기온 데이터 자동 로딩**
- ✅ **같은 형식 CSV 업로드 시 자동 전환**
- ✅ 선택한 날짜가  
  → 과거 **같은 월–일** 평균/중앙값 대비  
  → 얼마나 추웠거나 더웠는지(편차, 퍼센타일) 계산
- ✅ **Plotly 기반 인터랙티브 시각화**
  - 분포(Box plot)
  - 연도별 추이(Line chart)
  - 기준값 대비 비교(Bar chart)
- ✅ 결측치 자동 제외 (실측값만 사용)
